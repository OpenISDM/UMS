### URI Management Service (UMS) API ###

# Functionality Description
create URIs
delete URIs
search URIs

# Directory Structure
    .
    ├── config.xml
    ├── lib/
    │   ├── config.php
    │   └── open-source/
    │       ├── Copyright.txt
    │       └── LIB_http.php
    ├── readme.txt
    ├── UMS-GUI.html
    └── UMS.php

# Directory Structure Description
    config.xml      configuration file
    lib/            library directory
    config.php      configuration file class
    open-source/    open source code directory
    Copyright.txt   open source code copyright
    LIB_http.php    open source code
    readme.txt      document
    UMS-GUI.html    UMS GUI (functionality implemented by UMS.php API with HTTP POST method)
    UMS.php         UMS API
